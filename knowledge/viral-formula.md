# XiaoGi 病毒公式（Viral Formula）

## 核心方程式

```
[吉娃娃] + [與其體型身份不符的場景] = 視覺衝擊
```

狗的體型與品種本身就是整個梗的載體。圖片必須完全認真執行。文案強化他的認真感。

---

## 場景分類庫（Incongruity Taxonomy）

### A類：權威人物（Authority Figures）— 預期最高表現
XiaoGi 是負責人。場景暗示他有權力、責任或專業。

| 場景 | 關鍵視覺元素 |
|------|------------|
| 主持董事會 | 長型會議桌、皮革椅、身後投影幕 |
| 工地監工 | 安全帽、藍圖或平板、背景工人 |
| 站在講台發言 | 麥克風、講稿、前方觀眾席（暗示） |
| 法官席上 | 法槌、法袍（縮小版）、木質書桌 |
| 主廚審菜 | 廚師帽、白衣、人類端著盤子呈上 |
| 審訊室偵探 | 桌上聚光燈、案件資料夾、椅子 |

### B類：專業服務角色（Service Roles）— 中高表現
XiaoGi 正在完全專業地執行一份技術工作。

| 場景 | 關鍵視覺元素 |
|------|------------|
| 高級餐廳外場服務 | 白色桌布、小筆記本、正式制服 |

| 急診室醫生 | 聽診器、白袍、病歷夾 |

### C類：高地位休閒（Leisure as Expert）— 中等表現
XiaoGi 在高端休閒場所，以資深行家姿態出現，不是觀光客。

| 場景 | 關鍵視覺元素 |

### D類：日常官僚苦難（Domestic Bureaucracy）— 高共鳴（黑馬類）
XiaoGi 在面對枯燥的成年人生活，帶著明顯的壓力或嚴肅感。

| 場景 | 關鍵視覺元素 |
|------|------------|

| 超市仔細看產品背標 | 超市貨架、產品包裝、認真表情 |
| 講電話在等待音樂 | 電話貼耳朵、明顯不耐煩 |
| 在 ATM 看餘額 | ATM 螢幕、表情凝重 |

## 讓場景有效的要素

1. **一眼就懂**：觀看者在 1 秒內理解情境
2. **XiaoGi 是主角，不是道具**：他在做事，不是被放在那邊
3. **場景有人類尺度細節**：椅子、桌子、工具——暗示體型反差
4. **光線自然/專業**：不是工作室寵物攝影風格
5. **場景中只有 XiaoGi**：不要有其他動物

## 殺死場景的要素

- 場景過於複雜（超過 2 個焦點元素）
- XiaoGi 看起來害怕、困惑或「格格不入」——他永遠是主控方
- 可愛配件（蝴蝶結、項圈、派對帽）——除非被場景設定反轉
- 抽象或模糊背景——永遠要是真實可辨認的環境

---

## 場景輪換規則

- 不能連續兩天使用同一分類
- 14 天內不重複同一場景
- 累積 5 筆以上數據後，偏向 performance-log.md 中高表現的分類
- 每 30 天審視一次，清除已使用超過 3 次且表現平庸的場景
